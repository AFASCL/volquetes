# AI Log — Volquetes

Este directorio registra el uso de IA (Cursor + agentes) para trazabilidad del proyecto final.

## Estructura
- ai-log/prompts/: prompts usados + input/output + decisión humana
- ai-log/decisions/: ADRs o decisiones técnicas importantes

## Regla
Cada vez que se use IA para:
- PRD / backlog / arquitectura / diseño DB / implementación importante se agrega una entrada en prompts/ con fecha y agente.
